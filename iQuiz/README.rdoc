This application is written in Ruby 2.0 using Rails 4 framework.

<h2>Requirements</h2>
The user is presented with a true or false questions. 
A user can answer and add questions, but cannot answer a question that he has added.
The application will display statistics on each question.
<h2>Objective</h2>
This app was created in 4 days as the first exercise to implement a Rails application.
